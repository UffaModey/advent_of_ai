# Day 3: Hot Cocoa Championship Visualization Project

## Project Goal
Create presentation-ready visualizations for the Winter Festival Hot Cocoa Championship using Goose Auto Visualiser.  
We will visualize tournament progression, voting trends, and recipe comparisons.

---

## Step 1: Load the Tournament Data
Fetch the project data the from `hotcocoa_results.txt`
---

## Step 2: Visualizations to Create

### 2.1 Tournament Bracket
- Build a **flow chart** showing:
  - Quarterfinals → Semifinals → Championship
  - Contestant names and vote counts
  - Highlight the winner (Dark Chocolate Decadence)

### 2.2 Vote Distributions
- Create a **bar chart** showing:
  - Total votes per contestant in each round
  - Voting trends by period (morning, afternoon, evening)
- Highlight closest match and biggest blowout

### 2.3 Recipe Comparison
- Create a **radar chart** comparing recipe attributes (Richness, Sweetness, Creativity, Presentation)
- Each recipe should have a separate line
- Highlight the winner visually

### 2.4 Optional Extra Insights
- Chart the total votes over time periods
- Identify the most controversial recipe visually

---

## Step 3: Output Requirements
- Charts should be **presentation-ready** (PNG)
- Include:
  - Titles
  - Axis labels
  - Legends
- Make charts clear and visually engaging for the awards ceremony
