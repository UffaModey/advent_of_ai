# Festival Poster Generator

A Goose recipe that generates customized, polished festival posters with conditional styling based on event type. Perfect for creating professional-looking posters for any festival event.

## Features

- **Smart Conditional Logic**: Automatically styles posters based on event type
- **Professional Design**: Print-ready HTML with embedded CSS
- **Responsive Layout**: Works on both screen and print (8.5" x 11")
- **Event-Specific Customization**: Optional parameters for enhanced customization
- **Cross-Platform**: Works in both CLI and Desktop versions of Goose

## Usage

### Via Goose Desktop (Recommended)
1. Open the recipe in Goose Desktop:
   ```bash
   goose recipe open festival_poster_generator.yaml
   ```
2. Fill in the required parameters in the UI
3. Add optional parameters as needed
4. Generate your poster

### Via CLI
1. Open a new Goose session in this directory
2. Use the deeplink generated by:
   ```bash
   goose recipe deeplink festival_poster_generator.yaml
   ```

## Parameters

### Required Parameters
- **event_name**: Name of the festival event
- **event_datetime**: Date and time (e.g., "Saturday, December 21st at 7:00 PM")
- **location**: Event location/venue
- **event_type**: Choose from: `food`, `kids`, `performance`, `competition`, `workshop`

### Optional Parameters (Conditional)
- **local_delicacy**: *(food events only)* Special local food to showcase
- **theme**: *(kids events only)* Theme like "Pirates", "Dinosaurs", "Space Adventure"
- **industry**: *(workshop events only)* Field like "Technology", "Arts & Crafts", "Cooking"
- **level**: *(workshop events only)* Skill level like "Beginner", "Intermediate", "Advanced"
