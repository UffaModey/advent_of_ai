# Festival Closing Checklist

## Pre-Closing Announcements (30-60 minutes before close)
- [ ] Final performance/activity announcements
- [ ] Last call for vendor purchases
- [ ] Remind attendees of exit procedures
- [ ] Transportation information announced

## Event Close Procedures
- [ ] Final performance/activity completion
- [ ] Thank you and closing remarks
- [ ] Clear guidance on exit routes
- [ ] Begin controlled crowd dispersal

## Immediate Post-Close (0-2 hours)
- [ ] Secure cash registers and money handling areas
- [ ] Begin vendor breakdown coordination
- [ ] Security sweep of venue
- [ ] Lost and found collection
- [ ] Initial cleanup assessment
- [ ] Staff check-in and debriefing

## Breakdown Operations (2-6 hours post-close)
- [ ] Coordinate vendor equipment removal
- [ ] Sound/lighting system breakdown
- [ ] Remove temporary structures and barriers
- [ ] Collect all signage and decorations
- [ ] Waste management and recycling
- [ ] Return rented equipment

## Site Restoration (6-24 hours post-close)
- [ ] Deep cleaning of all areas
- [ ] Grounds restoration (if outdoor venue)
- [ ] Final security sweep
- [ ] Inventory check of remaining items
- [ ] Venue walkthrough with property management
- [ ] Final utilities shutdown

## Post-Event Administration (1-7 days)
- [ ] Financial reconciliation
- [ ] Incident report compilation
- [ ] Vendor feedback collection
- [ ] Staff performance reviews
- [ ] Thank you communications to partners
- [ ] Post-event analysis and lessons learned
- [ ] Archive event documentation
